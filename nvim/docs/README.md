# Documentation Guide

This directory contains comprehensive documentation for the Neovim LazyVim configuration.

## Directory Structure

### [guides/](./guides/)
Step-by-step guides and tutorials for different user levels:
- **beginner.md** - Getting started with Neovim
- **advanced.md** - Advanced configuration and customization
- **ai-integration.md** - Using AI assistants (OpenCode, Copilot)
- **troubleshooting.md** - Common issues and solutions

### [reference/](./reference/)
Quick reference documentation for fast lookup:
- **keymaps.md** - All keyboard shortcuts organized by category
- **plugins.md** - Plugin list with descriptions and usage
- **config.md** - Configuration options and variables
- **commands.md** - Custom commands and their usage
- **lsp.md** - LSP server configurations and capabilities

### [architecture/](./architecture/)
Technical and architectural documentation:
- **clean-architecture.md** - Design patterns and principles
- **project-structure.md** - Directory layout and organization
- **plugin-system.md** - How the plugin system works
- **lsp-setup.md** - LSP configuration details

1. **New User?** Start with [guides/beginner.md](./guides/beginner.md)

3. **Plugin question?** See [reference/plugins.md](./reference/plugins.md)
4. **Troubleshooting?** Look at [guides/troubleshooting.md](./guides/troubleshooting.md)
