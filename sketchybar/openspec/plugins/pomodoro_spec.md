# Especificación: Hacker Pomodoro (v3.1.0)

## REQUISITOS FUNCIONALES
- **Ciclos:** 25m (Trabajo) / 5m (Descanso corto) / 15m (Descanso largo tras 4 sets).
- **Iconos en Barra:**
    - 󰄉 (Inactivo / Idle)
    - 🍅 (Trabajo)
    - ☕ (Descanso)
    - 🏝️ (Descanso Largo)
- **Popup (Hover):**
    - Fase Actual (con emojis hacker: 👨‍💻, 🙌, 😴).
    - Progreso del Set (ej. 2/4).
    - Total de Pomos del día.
    - Botón de Reset.

## REQUISITOS TÉCNICOS
- **Transiciones:** Automáticas con notificaciones nativas.
- **Sonido:** Play sound al cambiar de fase.
- **Interacción:** Clic en barra para Play/Pausa. Clic en Reset para reiniciar ciclo.
